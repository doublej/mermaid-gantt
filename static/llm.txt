# Mermaid Gantt Brand Guidelines (LLM Reference)

> Concise brand specification for AI assistants generating Mermaid Gantt assets, code, or content.

## Identity

**Product:** Mermaid Gantt
**Tagline:** Visual Gantt charts from code
**Description:** Keyboard-first Gantt chart editor with Mermaid syntax import/export

## Logo

**Concept:** Stacked timeline bars with anchor node representing task dependencies and project flow. Built on golden ratio proportions.

**Construction (24x24 grid):**
- Bar 1: x=5, y=6, width=15, height=4, rx=1 (100% opacity)
- Bar 2: x=7, y=10, width=13, height=4, rx=1 (55% opacity)
- Bar 3: x=5, y=14, width=8, height=4, rx=1 (30% opacity)
- Anchor: circle at (5, 12), radius=2

**Minimum sizes:**
- 24px: Minimum usable
- 32px: Recommended
- 16px: Favicon only

**Clear space:** 4 grid units (1 bar height) on all sides

## Incorrect Usage (Don'ts)

- Don't rotate the logo
- Don't stretch or distort proportions
- Don't reduce opacity
- Don't use unapproved colors
- Don't add drop shadows or effects
- Don't place on busy backgrounds

## Motion

### Core Principles
1. **Swift & Courteous:** Fast entry, gentle settle; quick exit
2. **Purposeful:** Every animation communicates state change or feedback
3. **Consistent:** Same patterns use same timing/easing

### Easing Functions
- **Ease Out (primary):** cubic-bezier(0, 0, 0.2, 1) — Elements appearing, modals, tooltips
- **Ease In-Out:** cubic-bezier(0.4, 0, 0.2, 1) — Position changes, dragging, expanding
- **Ease In:** cubic-bezier(0.4, 0, 1, 1) — Elements exiting, closing
- **Linear:** Progress indicators, spinners only

### Duration Scale
- 100ms (Micro): Button press, icon change, checkbox
- 150ms (Fast): Tooltip, hover states
- 200ms (Standard): Default for most UI (modal open, panel expand)
- 300ms (Emphasis): Larger panels, complex reveals
- 500ms (Max): Page transitions, orchestrated sequences

### Stagger Pattern
- 50ms offset between items
- Top to bottom direction
- Cap at 8 items

### CSS Variables
```css
:root {
  --ease-out: cubic-bezier(0, 0, 0.2, 1);
  --ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-in: cubic-bezier(0.4, 0, 1, 1);
  --duration-micro: 100ms;
  --duration-fast: 150ms;
  --duration-base: 200ms;
  --duration-emphasis: 300ms;
  --duration-max: 500ms;
  --stagger-offset: 50ms;
}
```

### Reduced Motion
Always respect `prefers-reduced-motion: reduce`. Replace motion with instant state changes; opacity fades (≤100ms) can remain.
